//Розробники регламенту мають змогу видаляти UI-форми. Для цього виконайте наступні кроки:
Regulation developers have the ability to delete UI forms. To do so, follow these steps

. Open the *UI forms* section in the left-hand menu.
//. Відкрийте розділ "UI-форми" у меню зліва.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-1.png[]

. Select the context menu *Three dots* and click on `Delete`.
//. Оберіть контекстне меню "Три крапки" та натисніть `Видалити`.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-8.png[]

//NOTE: Після завершення ви побачите сповіщення про успішне видалення.
NOTE: After completion, you will receive a notification of the successful deletion.
