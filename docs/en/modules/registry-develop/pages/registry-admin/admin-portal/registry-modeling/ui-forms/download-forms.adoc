= Downloading UI forms
include::platform:ROOT:partial$templates/document-attributes/default-set-en.adoc[]

include::platform:ROOT:partial$admonitions/language-en.adoc[]

Regulation developers have the ability to download UI forms in JSON format for further processing in a local environment.

To download a form, follow these steps:

. Open the *UI forms* section in the left-hand menu.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-1.png[]

. Click on the _download icon_ &#10515;
+
image:registry-admin/admin-portal/ui-forms/ui-forms-10.png[]

NOTE: After successfully downloading the form, you will see a corresponding notification.

image:registry-admin/admin-portal/ui-forms/ui-forms-11.png[]
