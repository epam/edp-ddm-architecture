Registry developers can easily create UI forms for business processes.

[NOTE]
====
To start creating a form, first determine the version of the regulation.

At present, developers can create and edit forms in both master versions and candidate versions of the regulation.

For more information about versions of changes, please refer to the corresponding page
xref:registry-develop:registry-admin/admin-portal/version-control/version-control-overview.adoc[].
====

To create a form, you need to follow these steps:

. Open the *UI forms* section in the left-hand menu."
+
image:registry-admin/admin-portal/ui-forms/ui-forms-1.png[]

. Click on *Create new form*.
+
image:registry-develop:bp-modeling/forms/admin-portal-form-modeling-step-4.png[]

TIP: Learn more about the process of creating forms on the page xref:registry-develop:bp-modeling/forms/registry-admin-modelling-forms.adoc[].
