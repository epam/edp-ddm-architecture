Regulation developers can edit UI forms for business processes. To edit a form, follow these steps:

. Open the *UI forms* section in the left-hand menu.
//. Відкрийте розділ "UI-форми" у меню зліва.

//. Натисність _іконку редагування_ &#9998;
+
image:registry-admin/admin-portal/ui-forms/ui-forms-2.png[]

. Make the necessary changes to the components of the UI form and click `Save changes`.
//. Внесіть необхідні зміни до складових UI-форми та натисніть `Зберегти зміни`.
+
