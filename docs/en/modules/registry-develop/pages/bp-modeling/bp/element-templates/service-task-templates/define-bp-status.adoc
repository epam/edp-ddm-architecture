== General overview

//NOTE: _Перш за все, переконайтеся, що папка `/element-templates` містить файл `defineBusinessProcessStatusDelegate.json`._
NOTE: _Before proceeding, make sure that the `/element-templates` folder contains the `defineBusinessProcessStatusDelegate.json` file._
////
* Відкрийте **Service Task**, натисніть кнопку `Open Catalog` та оберіть шаблон (Template) зі списку.
* У полі `Name` вкажіть назву задачі.
* У полі `Status` вкажіть статус, що відображатиметься після завершення процесу.
////

* Open the **Service Task** and click the `Open Catalog` button to select a template from the list.
* In the `Name` field, specify the name of the task.
* In the `Status` field, specify the status to be displayed after the process is completed.
