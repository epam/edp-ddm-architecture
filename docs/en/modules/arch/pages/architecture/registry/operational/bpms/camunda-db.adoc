== Overview

//Призначенням бази даних `camunda` є зберігання даних про розгорнуті моделі бізнес-процесів та бізнес-правил, поточний стан та історичні дані виконання бізнес-процесів та інші службові дані, необхідні для роботи _Camunda Workflow Engine_.
The `camunda` database stores data about the deployed business process models and business rules, the current state and historical data of business process execution, and other service data necessary for the _Camunda Workflow Engine_ operation.
