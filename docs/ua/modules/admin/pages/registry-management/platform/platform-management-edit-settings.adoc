[#general-settings]
== Загальні налаштування

Розділ *Загальні налаштування* надає інструменти для керування налаштуваннями назви, логотипа і favicon Платформи, а також локалізацією інтерфейсу адміністративної панелі Control Plane, дозволяючи адміністраторам персоналізувати вигляд панелі керування.

image:registry-management/platform/cp-platform-mgmt-06.png[]

NOTE: Налаштування набудуть чинності після застосування запита на оновлення.

[name-logo]
[#name-logo]
=== Назва та логотип Платформи

Ця секція дозволяє адміністраторам Платформи кастомізувати назву та логотип їх інстансів через адміністративну панель Control Plane.
Ця функціональність підвищує рівень індивідуалізації та брендингу кожного інстансу, створюючи більш персоналізоване середовище для користувачів.

image:registry-management/platform/cp-platform-mgmt-06-1.png[]

Основні можливості ::

* *Кастомізація назви Платформи*: адміністратори можуть задавати унікальну назву для кожного інстансу Платформи.
* *Персоналізація логотипа*: можливість завантаження та налаштування кастомного логотипа для відображення в адміністративній панелі.
* *Налаштування Favicon*: вибір та встановлення власних favicon для інстансу Платформи, що забезпечує додатковий рівень брендингу.
* *Кастомізація Loader*: зміна зображення Loader для створення єдиного та інтегрованого візуального досвіду.

[TIP]
====
* *Header logo* (*логотип у заголовку*) -- це зображення або емблема, яке відображається у верхній частині сторінки чи програми, наприклад, на панелі навігації. Воно допомагає користувачам ідентифікувати бренд або платформу.

* **Favicon** (*значок сайту*) -- це маленька іконка, яка відображається на вкладці браузера поруч із назвою сайту. Вона допомагає швидко знайти потрібну вкладку серед інших.
====

.Опис параметрів налаштувань
[options="header"]
|===
| Параметр | Опис | Конфігурація

| Назва Платформи
| Введіть назву для вашої адміністративної панелі Control Plane, яка відображатиметься у лівому верхньому куті екрану разом із логотипом.
| Поле для редагування

| Зображення для логотипа (.svg)
| Завантажте файл логотипа у форматі `.svg`. Це зображення відображатиметься у лівому верхньому куті екрану разом із назвою Платформи.
| Поле для завантаження

| Зображення для favicon (32x32 px, .png)
| Завантажте файл у форматі `.png` із розміром 32x32 пікселі. Використовується як favicon у браузері.
| Поле для завантаження
|===

* Кастомізовані назва та логотип застосовуються до:

** Верхньої панелі навігації.
** Стартових сторінок адміністративної панелі.
** Відображення вікон завантаження (Loader).

* Після проходження пайплайну *cluster-mgmt*, усі зміни застосовуються без потреби повторної автентифікації.

=== Локалізація Платформи

Ця секція дозволяє адміністраторам Платформи налаштовувати мовні преференції для інтерфейсу адміністративної панелі Control Plane.
Доступні мови:

* UK-українська
* EN-англійська

image:registry-management/platform/cp-platform-mgmt-07.png[]

Основні можливості ::

* *Гнучкий вибір мови*: адміністратори можуть обирати між українською та англійською мовами для інтерфейсу Control Plane.

* *Локалізація інтерфейсу*:

** Повний переклад статичного контенту, плейсхолдерів, спливних вікон, повідомлень про помилки, підказок та довідників.
** Автоматичне застосування обраної мови до всіх відповідних елементів інтерфейсу.

* *Автоматичне встановлення мови за замовчуванням*: мова за замовчуванням наслідується з налаштувань регіону обслуговування, встановлених під час розгортання Платформи.
+
TIP: Див. детальніше про регіональні особливості розгортання у розділі xref:admin:installation/platform-deployment/platform-aws-deployment.adoc#deploy-platform-installer-scratch[Розгортання Платформи з Інсталятора].

.Опис параметрів налаштувань
[options="header"]
|===
| Параметр | Опис | Конфігурація

| Мова інтерфейсу
| Виберіть мову для інтерфейсу Control Plane. Доступні опції включають українську та англійську мови.
| Випадний список

| Регіональний формат
| Показує регіональний формат, який відповідає вашим мовним преференціям або регіону обслуговування. Змінюється автоматично, залежно від обраної мови інтерфейсу. Параметр дозволяє адаптувати відображення дат і часу.
| Автоматична, залежить від обраної мови інтерфейсу
|===

Натисніть *Підтвердити* для створення запита на оновлення.

[#backups]
== Резервне копіювання

Секція Резервне копіювання дозволяє налаштувати розклад для автоматичного створення резервних копій основних інфраструктурних компонентів Платформи, зокрема наразі це:

* *Nexus* -- центральне сховище артефактів.
* *Control Plane* -- адміністративна панель керування Платформою та реєстрами.
* *User Management* -- центральний компонент для управління автентифікацією та доступом користувачів.
* *Monitoring* -- центральний компонент для здійснення операцій моніторингу на Платформі.

image:registry-management/platform/cp-platform-mgmt-10.png[]
