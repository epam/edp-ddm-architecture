= Налаштування власних DNS-імен

Цей розділ містить деталізовані інструкції щодо конфігурації власних DNS-імен у реєстрі.

Налаштування власних DNS-імен дозволяє користувачам підключати власні доменні імена
до порталів користувачів і сервісу авторизації Keycloak,
надаючи зручність і брендування під час доступу до Платформи.
