Функція вибору мови дозволяє вам динамічно обирати мову інтерфейсу через селектор, розміщений у верхній частині вебінтерфейсу. Доступні мови в селекторі формуються на основі налаштування *`global.supportedLanguages`* у конфігурації платформи.

== Доступність та налаштування мов

Селектор мови знаходиться у верхній частині інтерфейсу адміністративного порталу. Він доступний на кожній сторінці й дозволяє вам обрати одну з підтримуваних мов. Перелік підтримуваних мов формується на основі налаштувань платформи, зокрема параметра `global.supportedLanguages`.

image:control-plane/platform/localization/cp-localization-22.png[]

== Збереження вибраної мови

Обрана мова зберігається у cookie з назвою `Language`, яка автоматично оновлюється при зміні мови в селекторі. Вибрана мова застосовується в реальному часі для оновлення інтерфейсу.

== Автоматичне визначення мови при завантаженні порталу

При завантаженні порталу система перевіряє наявність і значення cookie `Language`:

* Якщо cookie існує і її значення відповідає одній із підтримуваних мов, ця мова автоматично обирається в селекторі й застосовується для інтерфейсу.
* Якщо cookie відсутня або її значення не відповідає підтримуваній мові, система створить нову cookie `Language` зі значенням мови за замовчуванням (визначеним параметром `global.language`) і застосує цю мову.

== Застосування мови в реальному часі

Зміни у виборі мови застосовуються негайно без необхідності перезавантаження сторінки. Це забезпечує безперервний користувацький досвід із моментальним оновленням мови інтерфейсу.

image:control-plane/platform/localization/cp-localization-22-en.png[]

== Збереження вибору мови для майбутніх сесій

Вибрана мова зберігається в cookie `Language`. При наступному вході в систему інтерфейс автоматично відображатиметься мовою, обраною раніше.
