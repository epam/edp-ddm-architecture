= Вкладки на сторінці редагування форми

Сторінка редагування форми доступна за кліком на назві форми, або на іконці редагування (&#128393;). Для редагування форми, функціональність сторінки розділена на наступні вкладки:

== Загальна
Тут користувачеві доступні поля для заповнення бізнес-назви та службової назви форми. Бізнес-назву можна змінювати без обмежень, а службова назва формі дається один раз під час створення, і більше не змінюється.

image:registry-admin/admin-portal/ui-forms/main.png[]

== Код
В цій вкладці формується код форми, який можна редагувати та копіювати. Якщо форма створюється у **Конструкторі**, то код заповнюється автоматично.

image:registry-admin/admin-portal/ui-forms/code.png[]

== Конструктор
Ця вкладка дозволяє користувачеві створювати форму з елементів у UI. Елементи знаходяться у меню ліворуч, звідки їх можна перетягнути у робочу зону, щоб сформувати форму. Пошукове поле дозволяє відшукати потрібний елемент за назвою, або частиною назви.

image:registry-admin/admin-portal/ui-forms/constructor.png[]

== Перегляд
Тут користувач може перевірити, як виглядає та працює створена ним форма. Вкладка емулює роботу форми, щоб можна було перевірити всі її елементи.

image:registry-admin/admin-portal/ui-forms/view.png[]

== Запит
На цій вкладці знаходиться код запиту форми до зовнішнього API. Цей код можна скопіювати, але редагується він через певні зміни у вкладках **Конструктор** чи **Код**.

image:registry-admin/admin-portal/ui-forms/request.png[]

== Меню &#8942;
Окрім вкладок на сторінці є меню, що відкривається кліком на іконку "&#8942;". У цьому меню є наступні пункти:

* [*] Скопіювати форму. Цей пункт дозволяє створити копію даної форми, і почати редагувати її зі вкладки **Загальна**;
* [*] Експорт `.json`. Цей пункт дозволяє експортувати код форми у форматі `.json`;
* [*] Видалити. Цей пункт дозволяє видалити форму.
