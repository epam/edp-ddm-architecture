Розробники регламенту можуть здійснювати пошук UI-форм.
Пошук спрацьовує динамічно за одним з параметрів:

* назвою форми;
* службовою назвою форми.

Шукати змодельовані UI-форми на інтерфейсі можна так:

. Відкрийте розділ "UI-форми" у меню зліва.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-1.png[]

. Введіть назву форми в пошуковому вікні
+
image:registry-admin/admin-portal/ui-forms/ui-forms-4.png[]

NOTE: Пошук спрацьовує при введенні від 3 і більше символів.
