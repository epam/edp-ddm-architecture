Розробники регламенту можуть редагувати UI-форми для бізнес-процесів. Щоб редагувати форму, виконайте наступні кроки:

. Відкрийте розділ "UI-форми" у меню зліва.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-1.png[]

. Натисніть _іконку редагування_.
&#9998;
+
image:registry-admin/admin-portal/ui-forms/ui-forms-2.png[]

. Внесіть необхідні зміни до складових UI-форми та натисніть `Зберегти зміни`.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-3.png[]

TIP: Ознайомтеся детальніше зі складовими UI-форми на сторінці
xref:registry-admin/admin-portal/registry-modeling/ui-forms/form-tabs.adoc[]
