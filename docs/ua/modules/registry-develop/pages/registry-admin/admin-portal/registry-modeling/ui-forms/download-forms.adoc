Розробники регламенту мають змогу завантажувати (download) UI-форми у форматі JSON для подальшого опрацювання в локальному середовищі.

Щоб завантажити форму, виконайте наступні кроки:

. Відкрийте розділ "UI-форми" у меню зліва.
+
image:registry-admin/admin-portal/ui-forms/ui-forms-1.png[]

. Натисність _іконку завантаження_ &#10515;
+
image:registry-admin/admin-portal/ui-forms/ui-forms-10.png[]
