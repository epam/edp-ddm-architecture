= Підпроцеси

include::platform:ROOT:partial$admonitions/language-ua.adoc[]

Ця секція знайомить нас із підпроцесами в рамках моделювання бізнес-процесів за BPMN (Business Process Model and Notation). Підпроцеси є незамінними для декомпозиції складних процесів на більш керовані, менші частини. Ось, що ви дізнаєтесь:

* [*] *Вбудовані підпроцеси (Embedded subprocesses):* вони містяться в межах батьківського процесу і виконуються як частина загального потоку процесу.
* [*] *Виклик активностей (Call activity):* виклик активностей дозволяє повторно використовувати інші процеси або підпроцеси, сприяючи модульності та ефективності.
* [*] *Подієві підпроцеси (Event subprocesses):* вони активізуються специфічними подіями, додаючи гнучкості та оперативності у керуванні процесами.
* [*] *Транзакційні підпроцеси (Transaction subprocesses):* моделюють транзакції таким чином, що забезпечує успішне завершення всіх кроків у межах підпроцесу або їх повне скасування, підвищуючи надійність і консистентність.

Розуміння підпроцесів дозволить вам ефективно структурувати ваші бізнес-процеси в BPMN, сприяючи покращенню організації, масштабованості та точності виконання.
