* [*] Навчитись застосовувати обмеження на завантаження як для групи файлів, так і для окремих файлів

== Передумови

== Процес виконання завдання
=== Застосування обмеження на завантаження цифрових документів

Для застосування обмежень на завантаження цифрових документів слідуйте наступним крокам:

. Перевірте поточні обмеження. Для цього у редагуванні реєстру перейдіть на вкладку +++<b style="font-weight: 600">Цифрові документи</b>+++.
+
image:registry-develop:registry-admin-study/task-set-restriction-on-downloading/01-edit-registry.png[]
. Застосуйте наступні обмеження:
* Максимальний розмір файлу для завантаження, MB: *20*
* Максимальний сумарний розмір групи файлів для завантаження, MB: *80*
. Підтвердьте відповідні зміни.

[NOTE]
Результатом цього підзавдання буде встановлення лімітів для файлів.

image:registry-develop:registry-admin-study/task-set-restriction-on-downloading/02-set-values.png[]
