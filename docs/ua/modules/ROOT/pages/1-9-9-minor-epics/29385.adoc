== Загальні відомості

Коли Платформа і всі реєстри на ній створені з параметром регіону обслуговування `global`, для накладання та перевірки цифрового підпису користувачів кабінетів використовується *stub-віджет*.

image::platform:ROOT:1-9-9-minor-epics/29385/1.png[]

== Використання цифрового підпису

Можливість накладання та перевірки цифрового підпису залежить від конкретного регіону і його законодавчих вимог. Реалізація цієї функціональності для інших регіонів потребує адаптації відповідно до законодавства окремого регіону.

[stub-widget]
=== Stub-віджет

У випадку регіону Global, stub-віджет відображає статичний контент на формі – попередження та кнопку "Підписати дані".

* Попередження: Інформує користувача про обмеження функціональності.
* Кнопка "Підписати дані": При натисканні на кнопку не відбувається валідація даних ініціатора бізнес-процесу та даних з токена користувача.
