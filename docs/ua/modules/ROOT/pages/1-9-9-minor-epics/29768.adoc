= Застосування значень за замовчуванням для налаштувань залежно від обраного регіону обслуговування для Платформи та реєстрів

При розгортанні нової Платформи, в залежності від обраного адміністратором регіону -- `GLOBAL` або `UA`, за замовчуванням застосовуються відповідні значення для параметрів:

* _мова Платформи_ (`PLATFORM_LANGUAGE`) -- en або uk,
* _зображення для логотипа та лоадера_ -- глобус або герб України, відповідно.

Під час створення Платформи, адміністратор має можливість вручну змінити мову та зображення для неї.
