== Загальні відомості

У кабінетах користувачів, для реєстрів, створених з параметром *PLATFORM_REGION="global"*, відсутня можливість налаштувати канал зв'язку "Дія" для відправки повідомлень.

== Профіль користувача

Для реєстрів з параметром *PLATFORM_REGION="global"*, в особистій інформації профілю відображається лише ПІБ користувача.
