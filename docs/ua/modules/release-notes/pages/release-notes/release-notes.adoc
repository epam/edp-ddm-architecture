== Автоматична публікація REST API-документації сервісів до загальної документації проєкту (Antora)

[TIP]
====

Основні зміни: ::
+
* [*] ⚙️ *Автоматизація публікації REST API-документації*:
+
Тепер при будь-яких змінах у master/main гілці ваших сервісів, документація REST API автоматично публікується у загальну документацію проєкту. Це забезпечує актуальність та синхронізацію інформації.

* [*] ✍️ *Підтримка ручного оновлення документації*:
+
Для сервісів з ручним управлінням документації, ви можете додати файл REST API документації безпосередньо у репозиторій сервісу для її оновлення.

* [*] 🔍 *Розширений опис API*:
+
Введено детальніші описи методів, параметрів та відповідей у документації, що забезпечує глибше розуміння ваших API.

* [*] 📊 *Приклади відповідей API*:
+
Документація тепер містить зразки відповідей API, що допомагає у швидшому засвоєнні та тестуванні API.

* [*] 🔐 *Деталізація правил авторизації*:
+
Оновлено розділ про правила авторизації, з посиланнями на відповідні ресурси _Зовнішнього API-шлюзу операційної зони_ та _Підсистеми управління міжсервісною взаємодією_.

* [*] 🌐 *Єдиний мовний стандарт для API-документації*:
+
Раніше описи ендпоінтів наших API-специфікацій були представлені виключно українською мовою та не мали єдиного стандарту. Тепер ми внесли зміни, щоб уся API документація, незалежно від її первинної мови, відповідала єдиному міжнародному стандарту та була представлена англійською мовою. Це полегшує розуміння та використання API на міжнародному рівні та сприяє кращій інтеграції наших систем.

{empty}

* [*] 🛠️ Покращено Build-пайплайн для автоматичного пошуку та коміту Swagger-файлів у `ddm-architecture`.

== Підтримка декількох мов інтерфейсу перегляду звітів для посадових осіб (Redash-Viewer) та інтерфейсу для створення та налаштування аналітичних звітів та дашбордів (Redash-Admin)

[TIP]
====

Основні зміни: ::
+
* [*] 🌐 *Відображення інтерфейсів Redash Viewer та Redash Admin*:
+
Впроваджено можливість відображення інтерфейсів на мові, обраній адміністратором реєстру (українська або англійська).
* [*] *🔁 Розширення Replacement-скрипту для української мови*:
+
Вдосконалено переклади, особливо у частинах, які раніше не були локалізовані.

{empty}

Деталі: ::
+
* [*] 🔧 *Застосування обраної мови в Control Plane для інтерфейсів Redash*:
+
Забезпечено, що мова, вибрана в Control Plane для реєстру, автоматично застосовується до інтерфейсів Redash Admin та Redash Viewer.

* [*] 🌐 *Повна локалізація елементів інтерфейсу в Redash*:
+
Усі елементи інтерфейсу Redash були повністю локалізовані, щоб забезпечити зручність користувачів.

* [*] 💡 *Підказка про зміну мови в інтерфейсі Control Plane*:
+
Введено інформаційну підказку в адміністративній панелі Control Plane, що підкреслює, що зміна мови впливатиме на інтерфейси Redash.

{empty}

Загальний вплив: ::
+
📈 Це оновлення значно покращує досвід користування Redash, забезпечуючи більшу гнучкість та доступність для адміністраторів реєстру. Впровадження багатомовності забезпечує більшу адаптивність та зручність для користувачів, які працюють з різними мовними середовищами.
