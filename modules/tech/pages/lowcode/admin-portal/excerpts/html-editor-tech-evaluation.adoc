== Функціональні сценарії

[TIP]
Описати функціональні сценарії, які необхідно реалізувати в рамках дизайну рішення

- Внесення змін до HTML-шаблонів для генерації витягів з реєстру
- Внесення змін до HTML-шаблонів для генерації поштових повідомлень
- ...

== Приклади коду для редагування

[source, html]
----
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <div class="center">
        <img src="images/ua.png" class="trinity"/>
    </div>
    <p class="header text14" th:if="${user} == 'Joe'">
        ...
    </p>
    <table>
        <tbody>
            <tr th:each="animal: ${animals}">
              <td th:text="${animal.name}"></td>
            </tr>
        </tbody>
    </table>
</body>
</html>
----
